¿Qué es Django?

Django es un framework de alto nivel para desarrollar aplicaciones web con Python. Proporciona herramientas integradas para trabajar con bases de datos, plantillas, autenticación y administración.

Función de Django

Organiza la aplicación en componentes (Model, Template, View) y facilita tareas comunes: routing, gestión de datos, formularios, seguridad y despliegue.

Estructura típica

project_root/ ├─ manage.py ├─ project_name/ │ ├─ settings.py │ ├─ urls.py │ └─ wsgi.py └─ app_name/ ├─ models.py ├─ views.py ├─ urls.py ├─ templates/ └─ static/

Ventajas

  • Alta productividad y herramientas listas para producción.
  • ORM robusto, panel admin y seguridad por defecto.
  • Amplia comunidad y ecosistema de paquetes.

Desventajas

  • Puede ser más pesado que micro-frameworks en proyectos mínimos.
  • Aprender el ecosistema completo requiere tiempo.
